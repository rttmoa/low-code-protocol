## 基于文件的API

命令行和 VS Code同步工具使用此接口实现元数据双向同步。

### Deploy

批量上传元数据。

### DeployValidation

批量验证元数据。

### Retrieve

批量下载元数据。

## 基于 REST 接口的 API

实现单个元数据的增删改查。
